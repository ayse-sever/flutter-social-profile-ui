# Flutter Instagram Profile UI

This Flutter project Instagram profile screen user interface.
The screen includes a profile photo, post/follower/following counts, user description,
an "Edit Profile" button, a grid of posts, and bottom navigation tabs
(Home, Search, Add, Likes, Profile). No backend or interactivity is implemented; it focuses solely on UI design.

## Features
- Profile photo with post, followers, and following counts
- User description / status section
- "Edit Profile" button
- Grid layout for user posts
- Bottom navigation tabs: Home, Search, Add, Likes, Profile

## Technologies
- Flutter
- Dart
